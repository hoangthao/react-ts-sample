### Setup

#### Intialize project
```sh
yarn create vite .
yarn
yarn dev
```

```sh
yarn add @mantine/core @mantine/hooks @mantine/form @mantine/dates @tabler/icons-react @mantine/modals @mantine/notifications @mantinex/dev-icons

yarn add @hello-pangea/dnd
```
